*   TypeScriptのインストール
    ```
    npm install --save-dev typescript
    ```
*   @vue/cliのインストール
    ```
    npm install -g @vue/cli
    ```
*   Vue/cliでのプロジェクト作成
    ```
    vue create <プロジェクト名>
    ```
*   Viteでのプロジェクト作成
    ```
    npm init vite@latest
    ```
    -   プロジェクト名を入力
    -   vueを選択
    -   vue-tsを選択
